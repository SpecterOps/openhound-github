Traversable: ✅

## General Information

The traversable [GH_HasStep](/opengraph/extensions/githound/reference/edges/gh_hasstep) edge links a job to each of its steps in execution order. Created by `Parse-GitHoundWorkflow`, this edge enables analysts to enumerate all actions and shell commands executed by a job, including which secrets and variables each step consumes.
