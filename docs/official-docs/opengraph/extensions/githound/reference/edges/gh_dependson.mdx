## General Information

The non-traversable [GH_DependsOn](/opengraph/extensions/githound/reference/edges/gh_dependson) edge represents a `needs:` dependency between two jobs in the same workflow. Created by `Parse-GitHoundWorkflow`, this edge captures execution order constraints — the source job will not start until the destination job completes successfully. This edge is non-traversable because it represents sequencing only, not an access or privilege path.
