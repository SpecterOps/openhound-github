## General Information

The non-traversable [GH_ManageTopics](/opengraph/extensions/githound/reference/edges/gh_managetopics) edge represents a role's ability to manage repository topics used for discovery and classification. This permission is available to Maintain and Admin roles and custom roles that have been granted this specific permission.
