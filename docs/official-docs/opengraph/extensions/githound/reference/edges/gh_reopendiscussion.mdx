## General Information

The non-traversable [GH_ReopenDiscussion](/opengraph/extensions/githound/reference/edges/gh_reopendiscussion) edge represents a role's ability to reopen closed discussions to allow further replies. This permission is available to Triage, Write, Maintain, and Admin roles and custom roles that have been granted this specific permission.
