## General Information

The non-traversable [GH_DeleteDiscussion](/opengraph/extensions/githound/reference/edges/gh_deletediscussion) edge represents a role's ability to delete discussions. This permission is available to Triage, Write, Maintain, and Admin roles and custom roles that have been granted this specific permission.
